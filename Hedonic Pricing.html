import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error, r2_score
import numpy as np


df_train = df[0:train_size]
y_train = df_train['Price']
X_train = df_train.drop(columns=['Price'])

df_test = df[train_size:]
y_test = df_test['Price']
X_test = df_test.drop(columns=['Price'])

X_train['Mileage_Run'] = np.log1p(X_train['Mileage_Run'])
X_test['Mileage_Run'] = np.log1p(X_test['Mileage_Run'])

X_train['Mileage_Seating_Interaction'] = X_train['Mileage_Run'] * X_train['Seating_Capacity']
X_test['Mileage_Seating_Interaction'] = X_test['Mileage_Run'] * X_test['Seating_Capacity']

X_train = pd.get_dummies(X_train, drop_first=True)  
X_test = pd.get_dummies(X_test, drop_first=True)

X_test = X_test.reindex(columns=X_train.columns, fill_value=0)

complex_model = LinearRegression()
complex_model.fit(X_train, y_train)

y_train_pred_complex = complex_model.predict(X_train)
y_test_pred_complex = complex_model.predict(X_test)

rmse_train_complex = np.sqrt(mean_squared_error(y_train, y_train_pred_complex))
r2_train_complex = r2_score(y_train, y_train_pred_complex)

rmse_test_complex = np.sqrt(mean_squared_error(y_test, y_test_pred_complex))
r2_test_complex = r2_score(y_test, y_test_pred_complex)

print("Complex Model - Train RMSE:", rmse_train_complex, "R^2:", r2_train_complex)
print("Complex Model - Test RMSE:", rmse_test_complex, "R^2:", r2_test_complex)

Complex Model - Train RMSE: 138993.79352626577 R^2: 0.8528535357891419
Complex Model - Test RMSE: 156158.93760093872 R^2: 0.8356680844886679

This shows that, compared to the simpler model, the complex model performs better. It shows the connection between features and target variable more effectively, such as with the low RMSE and high R^2 values. This is why it's important to include trnasformations and the interactinos for better predictive performance. Certain categorical/numerical variables are likely to be predictors and a right skewed price distribution can better shape target variables. along with this, too many features can have a more complex model overfit if it captures the noise or small fluctuations in the training set creating poor performance in the test data.
